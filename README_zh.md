# 如何使用自动评分机

ok程序是伯克利的自动评分机
如果您不是伯克利的学生，则需要在命令中添加 --local以跳过输入电子邮件的过程
如：
把
---
python3 ok -q python-basics -u 
---
改成
---
python3 ok -q python-basics -u --local
---
